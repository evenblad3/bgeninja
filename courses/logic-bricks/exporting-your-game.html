Exporting your game as a .EXE

Exporting your game is as easy as pie. Goto File -> Export -> Save As Game Engine Runtime. If you don't find it, you can enable it by going to User Preferences by pressing Ctrl+Alt+U or by going to File -> User Preferences.

After choosing it, you will be shown a menu like this. This is to export your game in a directory.

UPBGE 2.6.010+

If you're using 2.6.010 you may find the option on the top. This was brought to make the steps simpler, thanks to the developer.

Range 1.4+
