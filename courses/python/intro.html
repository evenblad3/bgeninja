Introduction to game engine's API

You'll learn to code with UPBGE (or) Range API using Python. Covering from basic to intermediate level.

What you'll learn

  • Move objects
  • Manipulate objects in other scene
  • Play sounds without logic bricks
  • Create properties and debug them without touching the UI
  • Get framrate, object speed, OpenGL Info
  • bricks/python is easy stuff vs all bricks setups.
  • and more!
